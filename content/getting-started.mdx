# Installation

@chaesunbak/registry 레지스트리의 컴포넌트를 사용하기 위해서는 우선 `components.json`에 레지스트리 주소를 추가해야 합니다.


## Command

`shadcn` CLI를 사용하여 추가하거나

```bash
npx shadcn@latest add https://registry.chaesunbak.com/r/{name}.json
```

## Manual

`components.json`에 다음과 같이 레지스트리 주소를 추가하세요.

```json
{
  "registries": {
    "@chaesunbak": "https://registry.chaesunbak.com/r/{name}.json"
  }
}
```
