## Manual

`components.json`에 다음과 같이 레지스트리 주소를 추가하세요.

```json
{
  "registries": {
    "@chaesunbak": "https://registry.chaesunbak.com/r/{name}.json"
  }
}
```

> 노트: `{name}` 플레이스홀더는 npx shadcn@latest add @chaesunbak/button 명령어를 실행할 때 자동으로 리소스 이름으로 파싱되어 대체되므로 수정할 필요없습니다.
