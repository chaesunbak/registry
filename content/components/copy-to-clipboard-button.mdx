## 인터페이스

### CopyToClipboardButton

| 속성        | 타입         | 기본값      | 설명                       |
| ----------- | ------------ | ----------- | -------------------------- |
| `text`      | `string`     | -           | 복사할 텍스트입니다.       |
| `className` | `string`     | `undefined` | 추가적인 스타일 클래스입니다. |
| `onSuccess` | `() => void` | `undefined` | 복사 성공 시 호출됩니다.   |
| `onError`   | `() => void` | `undefined` | 복사 실패 시 호출됩니다.   |
